Project Title: Responsive Eatery Ordering Website

Live link :https://nhambe-dokotela.github.io/Kwamanjobo-mobile-kitchen/ 

Description:

Developed a fully responsive frontend website for a kitchen/eatery to showcase business information and menu items. Implemented a WhatsApp order integration button that redirects customers directly to the business WhatsApp for order placement. The project emphasizes mobile-first design, accessibility, and user-friendly navigation for small business customers.

Key features:

Responsive layout for mobile and desktop

Business profile and menu display

WhatsApp order integration

Clean and simple UI

Technologies:

HTML5, CSS3, Responsive Web Design
